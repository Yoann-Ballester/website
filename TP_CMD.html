TPs CMD

TP-1

Un fichier batch : militaire.bat (contenu disponible ci-dessous) permettant d'avoir un retour "Affirmatif" si interroger de OUI, "negatif" si interroger de NON ou "erreur" pour le reste la casse/majuscules étant à respecter ici.

@echo off
if "%1"== "NON" goto negatif
if "%1"== "OUI" (goto affirmatif) else (echo erreur)
goto end
:affirmatif
echo Affirmatif
goto end
:negatif
echo negatif
goto end
:end

TP-2

Création de fichiers batch permettant de créer et supprimer une arborescence de dossiers et fichiers utilisable partout sur la machine grâce à la création de variable d'environnement tenant compte du chemin des fichiers batch.

Ci-dessous le fichier batch permettant de créer l'arborescence de dossiers et fichiers.

mkdir repertoire1\r5 r2 r3 r4\r6 r4\r7 r4\r8
cd repertoire1
type nul > fichier1.txt
type nul > f2.txt
cd ..\r4\r8
type nul > f3.txt
cd ..
cd ..

Ci-dessous le fichier batch permettant de supprimer l'arborescence de dossiers et fichiers crée par la premier fichier batch.

rmdir repertoire1 r4 r3 r2 /s /q
